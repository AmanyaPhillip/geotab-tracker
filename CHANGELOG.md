### Added
- Real-time vehicle tracking on an interactive Leaflet map.
- Vehicle selector with live position updates via the Geotab API.
- G-force and trip charts rendered with Recharts.
- Past-trips card for reviewing historical journeys.
- Auth form and an error boundary for resilient loading.
- GitHub Pages deployment via `gh-pages`.
